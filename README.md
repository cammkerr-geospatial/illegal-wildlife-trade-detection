\# Wildlife Trade Detection System



Automated system for detecting wildlife trade discrepancies using CITES trade data and IUCN Red List classifications.



\## Objectives



\- Detect mismatches between exporter and importer trade reports

\- Identify wildlife trade hubs

\- Map international wildlife trade networks

\- Generate GIS-ready outputs for QGIS



\## Data Sources



\- CITES Trade Database

\- IUCN Red List
